# Tofu-utils

> The util collection for Tofu.

## Usage

~~~javascript
import { Num } from 'tofu-utils'

Num.plus(1, 2) // 3
Num.reduct(2, 1) // 1
Num.multiply(1, 2) // 2
Num.divide(2, 2) // 1
~~~

## Develope

`npm run dev`

## Build

`npm run build`

## Test

`npm run test`

## Publish

`npm run issue`
